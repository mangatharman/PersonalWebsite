HTML Cheat Sheet

Introduction Basics
  • <html> = Root element of HTML Page
  • <head> = Contains meta information about the HTML Page
  • <title> = Specifies a title for the HTML Page
  • <body> = Is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • <h1> = Defines a large heading
  • <p> = Defines a paragraph

For more information: Click here

  • Headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading while <h6> defines the least important heading
  • Links are defined with the <a> tag
  • Images are defined with the <img> tag

For more information: Click here
Elements Attributes
  • The HTML element is everything from the start tag to the end tag
  • HTML elements can be nested. For example, using <body> tag inside the <html> tag and using <p> and <h1> inside the <body> tag
  • Never skip the end tags
  • Some elements don't have an end tag. For example, <br> defines a line break and it does not have an end tag
  • HTML is not case sensitive. For example, <p> means the same as <P>

For more information: Click here
  • Attributes provide additional information about the elements
  • Attributes are always specified in the start tag
  • The href attribute in the <a> tag specifies the URL of the page the link goes to
  • The src attribute in the <img> tag specifies the path to the image to be displayed
  • The width and height attribute in the <img> tag specifies the width and height of the image
  • The style attribute is used to add style such as color, font, size, and more
  • The lang attribute declares the language of the web page
  • The title attribute defines some extra information about an element
  • Always use lowercase attributes and always quote attribute values

For more information: Click here
Styles Text Formatting
  • Style attribute is used for styling HTML elements
  • Use background-color to change the background color of a HTML page or a paragraph or anything
  • Use color to change the color of the text
  • Use font-family to change the font of the text
  • Use font-size to change the size of the text
  • Use text-align for the horizontal text alignment

For more information: Click here
  • <b> = Defines bold text
  • <em> = Defines emphasized text
  • <i> = Defines text in an alternate voice or mood
  • <small> = Defines smaller text
  • <strong> = Defines important text
  • <sub> = Defines subscripted text
  • <sup> = Defines superscripted text
  • <ins> = Defines inserted text
  • <del> = Defines deleted text
  • <mark> = Defines marked/highlighted text

Quotations Colors
  • <abbr> = Defines an abbreviation or acronym
  • <address> = Defines contact information for the author/owner of a document
  • <bdo> = Defines the text direction
  • <blockquote> = Defines a section that is quoted from another source
  • <cite> = Defines the title of a work
  • <q> = Defines a short inline quotation

For more information: Click here
  • HTML colors are specified with predefined color names, or with RGB, HEX, HSL Values
  • An RGB color value represents Red, Green and Blue light sources. For example, rgb(255, 0, 0) would display the color red
  • An RGBA color value is an extension of RGB with an Alpha channel (opacity).
  • A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.
  • HSL stands for hue, saturation, and lightness. For example, hsl(240, 100%, 50%) would display the color Blue
  • HSLA color values are an extension of HSL with an Alpha channel (opacity).

For more information: Click here
HTML CSS Links
  • Use the HTML style attribute for inline styling
  • Use the HTML <style> element to define internal CSS
  • Use the HTML <link> element to refer to an external CSS file
  • Use the HTML <head> element to store <style> and <link> elements

For more information: Click here
  • The target attribute specifies where to open the linked document
  • _self = Default. Opens the document in the same window
  • _blank = Opens the document in a new window
  • _parent = Opens the document in the parent frame
  • _top = Opens the document in the full body of the window
  • Absolute URL = A full web address in the href attribute
  • Relative URL = A link to a page within the same website
  • Put the <img> tag in the <a> tag to use an image as a link
  • Use mailto: inside the href attribute to create a link that opens the user's email program
  • To use HTML button as a link, we need to use some JavaScript code

Images Tables
  • <img> is an open tag and it does not require a closing tag
  • Use src attribute to define the URL of an image
  • Use alt attribute to define an alternate text for an image
  • Use the CSS float property to let the image float to the left or to the right
  • <map> = Defines an image map
  • <area> = Defines a clickable area inside an image map
  • <picture> = Defines a container for multiple image resources

For more information: Click here

  • <table> = Defines a HTML Table
  • <tr> = Table Row
  • <th> = Table header
  • <td> = Table cell/data
  • <caption> = Table Caption
  • Use the CSS border property to define a border, CSS border-collapse property to collapse cell borders
  • Use the CSS padding property to add padding to cells, CSS text-align property to align cell text
  • Use the CSS border-spacing property to set the spacing between cells
  • Use the colspan attribute to make a cell span many columns, Use the rowspan attribute to make a cell span many rows, Use the id attribute to uniquely define one table

Lists Block and Inline
  • <ul> = Defines an unordered List
  • <ol> = Defines an ordered List
  • <li> = Defines a list item
  • <dl> = Defines a description list
  • <dt> = Defines a term in a description list
  • <dd> = Describes the term in a description list

For more information: Click here

  • A block-level element always starts on a new line and takes up the full width available
  • An inline element does not start on a new line and it only takes up as much width as necessary
  • The <div> defines a section in a document (block-level)
  • The <span> defienes a section in a document (inline)

For more information: Click here

Classes ID
  • The HTML class attribute specifies one or more class names for an element
  • Classes are used by CSS and JavaScript to select and access specific elements
  • The class attribute can be used on any HTML element
  • The class name is case sensitive
  • Different HTML elements can point to the same class name
  • JavaScript can access elements with a specific class name with the getElementsByClassName() method

For more information: Click here

  • The id attribute is used to specify a unique id for an HTML element
  • The value of the id attribute must be unique within the HTML document
  • The id attribute is used by CSS and JavaScript to style/select a specific element
  • The value of the id attribute is case sensitive
  • The id attribute is also used to create HTML bookmarks
  • JavaScript can access an element with a specific id with the getElementById() method
